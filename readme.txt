КАК ДА ДЕКРИПТИРАТЕ ФАЙЛ В WINDOWS (ЛЕСНО РЪКОВОДСТВО)

Поставете програмата и криптирания файл в една папка.
Създайте папка, например:
C:\crypto
Вътре в нея поставете:

secure_file.exe (вашата програма)
libsodium-23.dll (няма да тръгне без него)

криптирания файл (encrypted.txt)

име за декриптирания изходен файл (пример: decrypted.txt)

Отворете Command Prompt в същата папка.
Най-лесният метод:

Отворете папката в File Explorer.

Кликнете върху адресната лента най-отгоре.

Напишете: cmd

Натиснете Enter.
Ще се отвори черен прозорец (Command Prompt), вече намиращ се в правилната папка.

Стартирайте командата за декриптиране.
Форматът е:
secure_file.exe dec <име_на_криптиран_файл> <име_на_изходен_файл>

Пример:
secure_file.exe dec encrypted.txt decrypted.txt

encrypted.txt е криптираният файл.
decrypted.txt е файлът, който ще бъде създаден след декриптиране.

Натиснете Enter, за да изпълните командата.

Въведете паролата.
Ще се появи:
Enter password:
Въведете паролата, използвана при криптирането, и натиснете Enter.

Ако паролата е грешна, ще видите:
Wrong password or corrupted file

Ако е вярна, ще видите:
File decrypted successfully.

Намерете декриптирания файл.
Файлът (например decrypted.txt) ще се появи в същата папка като .exe файла. Можете да го отворите нормално според неговия тип.
